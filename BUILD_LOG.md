**Entry template:**

```
## YYYY-MM-DD — <session focus> (milestone Mx)
**Did:** what was attempted.
**Result:** what passed / failed. Verification suite GREEN/RED (+ reason if RED).
**Decided:** any decision (→ ADR-NNNN if consequential).
**Open:** unresolved threads, surprises, things the next session should know.
**Next action:** the single concrete next step.
```
