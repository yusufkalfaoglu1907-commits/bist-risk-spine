```

**Decided:** any decision (→ ADR-NNNN if consequential).
```

**Did:** ran the §4 adversarial review — a fresh falsifying agent (no build context) attacked the M4 harness with the sole mandate of making it lie. - **D3 (low) — short history raised `ValueError` instead of failing cleanly.** `T < pbo_partitions` propagated an exception out of the gate. **Fix:** `_safe_pbo` clamps partitions to the largest even ≤ T and returns a NaN verdict when T<2 ⇒ the candidate fails `pbo_below_threshold` (rejected), never crashes.
